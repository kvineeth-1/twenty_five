Content-box = it is a default property of Box-sizing. If we set an element's width to 200 pixels, then the element's content box will be 200 pixels wide, and the width of any border or padding will be added to the final width and height, making the element wider than 200px.

>

Border-box = the elements width and height remains the same. If we set an element's width to 200 pixels, then the element's content box will be less than 200 pixels wide, and the width of any border or padding will be deducted to the final width and height, making the element less than 200px.
